Installation Guide

  1. 1
    Unlock Bootloader

    Enable OEM unlocking in Developer Options, then run fastboot flashing unlock.

  2. 2
    Flash Custom Recovery

    Download the recommended recovery for your device and flash via fastboot flash recovery recovery.img.

  3. 3
    Sideload pj.skr

    Boot into recovery, choose "Apply update → ADB sideload", then run adb sideload pjskr-*.zip.

  4. 4
    Reboot & Enjoy

    First boot may take a few minutes. Customise your setup and enjoy pj.skr!

Getting Started

New to custom ROMs? Start here.

  • Prerequisites — Unlocked bootloader, USB debugging enabled, ADB & Fastboot installed on your PC.
  • Backup — Always make a full TWRP / OrangeFox backup before flashing.
  • Download — Grab the latest pj.skr ZIP for your device from the Download tab or GitHub Releases.
🛠️

Building from Source

  1. Install dependencies: sudo apt install bc bison build-essential …
  2. Init repo: repo init -u https://github.com/pjskr/manifest -b sakura
  3. Sync: repo sync -c -j$(nproc) --force-sync --no-clone-bundle
  4. Set up env: source build/envsetup.sh
  5. Choose target: lunch pjskr_<codename>-userdebug
  6. Build: mka bacon -j$(nproc)

FAQ

Will I lose my data?
A clean flash wipes data. Dirty flash (same Android base) usually keeps data — but always backup.
Does SafetyNet / Play Integrity pass?
Yes, out of the box with our bundled attestation fix.
How do I get OTA updates?
Go to Settings → System → pj.skr Updater. Updates are pushed as soon as new builds are ready.
Where can I report bugs?
Open an issue in the device-specific repo inside pjskr-Devices.
⚙️

Kernel Info

  • Base — Latest CAF / GKI kernel for each SoC.
  • Scheduler — BORE scheduler for lower latency.
  • I/O — BFQ I/O scheduler, F2FS optimisations.
  • Governors — Schedutil by default, walt optional.
